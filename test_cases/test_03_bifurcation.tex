% Test Case 03: Bifurcation in a 1D ODE
% Difficulty: Hard — requires analysis/topology arguments

\begin{claim}
The ODE $\dot{x} = r x - x^3$ undergoes a pitchfork bifurcation at $r = 0$.
For $r \leq 0$ the only equilibrium is $x^* = 0$ (stable).
For $r > 0$ there are three equilibria: $x^* = 0$ (unstable) and $x^* = \pm\sqrt{r}$ (stable).
\end{claim}
